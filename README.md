## Scripts

**Development mode**

```bash
  yarn dev 
```
or 
```bash
  npm dev  
```

**Build**

```bash
  yarn build 
```
or 
```bash
  npm build  
```
## Funcionalidades

- Criação de Tarefas
- Eliminação de Tarefas
- Conclusão de Tarefas
